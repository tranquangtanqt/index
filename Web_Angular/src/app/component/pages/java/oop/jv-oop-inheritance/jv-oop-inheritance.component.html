Kế thừa (Inheritance)

  1. Trang chủ
  2. Java
  3. Kế thừa (Inheritance)

Kế thừa là quá trình cho phép một lớp có được các thuộc tính (phương thức và các biến) của một lớp khác. Với sự kế thừa, thông tin được đặt theo thứ tự có thứ bậc có thể quản lý được.

Lớp kế thừa các đặc tính của lớp khác là lớp con (subclass) (còn được gọi là lớp kế thừa); lớp mà các thuộc tính của nó được kế thừa là lớp cơ sở (hoặc lớp cha).

- Để kế thừa từ một lớp, sử dụng từ khóa extends (mở rộng).

Ví dụ dưới đây cho thấy làm thế nào để có lớp Dog được kế thừa từ lớp Animal.

class Dog extends Animal {{ '{' }}
 // some code
{{ '}' }}

Ở đây, Dog là lớp con còn Animal là lớp cha.

- Khi một lớp được kế thừa từ một lớp khác, nó thừa hưởng tất cả các biến và phương thức (trừ các phương thức private).

class Animal {{ '{' }}
    protected int legs;
    public void eat() {{ '{' }}
        System.out.println("Animal eats");
    {{ '}' }}
{{ '}' }}

class Dog extends Animal {{ '{' }}
    Dog() {{ '{' }}
        legs = 4;
    {{ '}' }}
    
    public void variable() {{ '{' }}
        System.out.println(legs);
    {{ '}' }}
    
{{ '}' }}

class MyClass {{ '{' }}
    public static void main(String[ ] args) {{ '{' }}
        Dog d = new Dog();
        d.eat();
        d.variable();
    {{ '}' }}
{{ '}' }}

/*********
Output
Animal eats
4
*********/

- Constructors của lớp cha được gọi khi lớp con được khởi tạo.

- Có thể truy cập lớp cha từ lớp con bằng cách sử dụng từ khóa super.

Ví dụ, super.var truy cập các thành viên var của lớp cha.

class A {{ '{' }} 
    int num1 = 50;
    public A() {{ '{' }} 
        System.out.println("New A");
    {{ '}' }}
{{ '}' }} 

class B extends A {{ '{' }} 
    int num1 = 100; 
    public B() {{ '{' }} 
        System.out.println("New B");
        System.out.println(num1); /*Lấy giá trị biến num1 trong lớp hiện tại nếu đã được khai báo. 
        Nếu  chưa được khai báo, nó sẽ lấy giá trị từ lớp cha.*/
        System.out.println(super.num1); /*Nếu bạn đã khai báo num1 trong cả A và B, 
        và bạn muốn lấy giá trị từ lớp cha, bạn sử dụng từ khóa super 
        và sẽ sử dụng giá trị từ lớp cha là 50*/
    {{ '}' }} 
{{ '}' }} 

class Program {{ '{' }} 
    public static void main(String[ ] args) {{ '{' }} 
        B obj = new B(); 
    {{ '}' }}
{{ '}' }}

/********
 * Output
 New A
 New B
 100
 50
 */
