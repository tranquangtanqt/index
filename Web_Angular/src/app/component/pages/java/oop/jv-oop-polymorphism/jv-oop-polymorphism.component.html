Đa hình, trong đó đề cập đến ý tưởng 'có nhiều hình thức', xảy ra khi có một thứ bậc của các lớp liên quan đến nhau thông qua kế thừa. (Đa hình là một khái niệm mà chúng ta có thể thực hiện một hành động bằng nhiều cách khác nhau).

Ví dụ: Dog và Cat là các lớp thừa hưởng từ lớp Animal. Mỗi lớp có thực hiện riêng một phương thức makeSound ().

class Animal {{ '{' }}
    public void makeSound() {{ '{' }}
        System.out.println("Grr...");
    {{ '}' }}
{{ '}' }}

//Dog và Cat là các lớp kế thừa từ lớp Animal. Mỗi lớp thực hiện riêng một phương thức makeSound () 
class Cat extends Animal {{ '{' }}
    public void makeSound() {{ '{' }}
        System.out.println("Meow");
    {{ '}' }}
{{ '}' }}

class Dog extends Animal {{ '{' }}
    public void makeSound() {{ '{' }}
        System.out.println("Woof");
    {{ '}' }}
{{ '}' }}

class Program {{ '{' }}
    public static void main(String args[ ]) {{ '{' }}
        //Tạo ra hai biến tham chiếu của Animal và chỉ chúng tới đối tượng Cat and Dog.
        Animal a = new Dog();
        Animal b = new Cat();
        
        a.makeSound();
        b.makeSound();
    {{ '}' }}
{{ '}' }}

/**********
 * Output
 * Woof
 * Meow
***********/
