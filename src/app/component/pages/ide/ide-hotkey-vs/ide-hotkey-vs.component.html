Hotkey Chức năng
F12 – See implementation Xem chi tiết một class hay method.
Ctrl R + Ctrl E Tạo nhanh get set
Ctrl K + Ctrl D Format Code.
Ctrl K + Ctrl C Comment một đoạn code
Ctrl K + Ctril U Bỏ comment một đoạn code
F5 – Ctrl F5 Debug code. Trường hợp đặt breakpoint để bug, hãy bấm Ctrl + F5 để chạy chế độ no debug, bỏ qua breakpoint
F9 Đặt nhanh breakpoint
F10/F11 Step over/Step into (Dùng trong debug)
Ctrl K + Ctrl S Tạo nhanh snippet - để tạo nhanh for, while, if, contructor...
Ctrl K + Ctrl X Tương tự Ctrl K + Ctrl S nhưng có thể chọn nhiều hơn cho ASP.Net, test, Visual...
Giữ Ctrl + click chuột Sửa nhiều dòng code một lúc sử dụng Box Selection
Ctrl L Xóa dòng hiện tại (tại điểm con trỏ chuột).
Shift + Alt + T Đổi chỗ dòng tại vị trí chuột với dòng dưới nó.
Ctrl + U Chuyển đổi các ký tự được bôi đen thanh chữ thường.
Ctrl + Shift + U Chuyển đổi các ký tự được bôi đen thường thành chữ in HOA.
