Anonymous class

  1. Trang chủ
  2. Java
  3. Anonymous class

Lớp Anonymous là một cách để mở rộng các lớp hiện tại một cách nhanh chóng.

Ví dụ: Hãy xem xét có một lớp Machine:

class Machine {{ '{' }}
  public void start() {{ '{' }}
    System.out.println("Starting...");
  {{ '}' }}
{{ '}' }}

Khi tạo đối tượng Machine, chúng ta có thể thay đổi phương thức start() ngay lập tức.

public static void main(String[ ] args) {{ '{' }}
  Machine m = new Machine() {{ '{' }}
    @Override public void start() {{ '{' }}
      System.out.println("Wooooo");
    {{ '}' }}
  {{ '}' }};
  m.start();
{{ '}' }}
class Machine {{ '{' }}
    public void start() {{ '{' }}
        System.out.println("Starting...");
    {{ '}' }}
{{ '}' }}

class Program {{ '{' }}
    public static void main(String[ ] args) {{ '{' }}
        Machine m = new Machine() {{ '{' }}
            @Override public void start() {{ '{' }}
                System.out.println("Wooooo");
            {{ '}' }}
        {{ '}' }};
        m.start();
    {{ '}' }}
{{ '}' }}

/*****************
 * Output
 * Wooooo
*****************/

Sau khi khai báo đối tượng, chúng ta đã mở ngoặc nhọn và đã ghi đè lên việc triển khai của phương thức start().

Chú thích: @Override được sử dụng để làm cho code dễ hiểu hơn, có thể bỏ đi nhưng đầu ra vẫn không thay đổi.

- Sửa đổi chỉ áp dụng đối với đối tượng hiện tại, chứ không phải là đối tượng của lớp. Vì vậy, nếu chúng ta tạo ra một đối tượng khác của lớp đó, việc thực hiện phương thức start() sẽ là một trong những định nghĩa của lớp ban đầu.

class Machine {{ '{' }}
    public void start() {{ '{' }}
        System.out.println("Starting...");
    {{ '}' }}
{{ '}' }}

class Program {{ '{' }}
    public static void main(String[ ] args) {{ '{' }}
        Machine m1 = new Machine() {{ '{' }}
            public void start() {{ '{' }}
                System.out.println("Wooooo");
            {{ '}' }}
        {{ '}' }};
        Machine m2 = new Machine();
        m2.start();
    {{ '}' }}
{{ '}' }}

/***************
 * Output
 * Starting...
***************/
