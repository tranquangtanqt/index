Getters & Setters

  1. Trang chủ
  2. Java
  3. Getters & Setters

- Getters và Setters được sử dụng để bảo vệ dữ liệu một cách hiệu quả, đặc biệt khi tạo các lớp. Đối với mỗi biến, phương thức get trả về giá trị, trong khi phương thức set đặt giá trị.

- Getters bắt đầu bằng get, tiếp theo là tên biến, với chữ cái đầu tiên của tên biến được viết hoa.

- Setters bắt đầu bằng set, theo sau là tên biến, với chữ cái đầu tiên của tên biến được viết hoa.

- Phương thức getter trả về giá trị của thuộc tính.

- Phương thức setter lấy một tham số và gán nó cho thuộc tính.

public class Vehicle {{ '{' }}
    private String color;
    
    // Getter
    public String getColor() {{ '{' }}
        return color;
    {{ '}' }}
    
    // Setter
    public void setColor(String c) {{ '{' }}
        this.color = c;
    {{ '}' }}
{{ '}' }}

class Program {{ '{' }}
    public static void main(String[ ] args) {{ '{' }}
        Vehicle v1 = new Vehicle();
        v1.setColor("Red");
        System.out.println(v1.getColor());
    {{ '}' }}
{{ '}' }}
/*** OUTPUT ***
Red
***************/
