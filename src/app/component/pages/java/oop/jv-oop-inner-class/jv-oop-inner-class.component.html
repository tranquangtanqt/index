Lớp lồng nhau (Inner Class)

  1. Trang chủ

  3. Lớp lồng nhau (Inner Class)

Java hỗ trợ các lớp lồng nhau; một lớp có thể là thành viên của một lớp khác.

Tạo một lớp bên trong một lớp là khá đơn giản. Chỉ cần viết một lớp trong một lớp. Không giống như một lớp, một lớp bên trong có thể là private. Một khi bạn khai báo lớp bên trong private, nó không thể được truy cập từ một đối tượng bên ngoài lớp.

class Robot {{ '{' }}
    int id;
    Robot(int i) {{ '{' }}
        id = i;
        Brain b = new Brain();
        b.think();
    {{ '}' }}

    private class Brain {{ '{' }}
        public void think() {{ '{' }}
            System.out.println(id + " is thinking");
        {{ '}' }}
    {{ '}' }}
{{ '}' }}

public class Program {{ '{' }}
    public static void main(String[] args) {{ '{' }}
        Robot r = new Robot(1);
    {{ '}' }}
{{ '}' }}

/******************
 * Output
 * 1 is thinking
******************/

Lớp Robot có một lớp Brain ở bên trong. Lớp bên trong có thể truy cập vào tất cả các biến thành viên và các phương thức của lớp bên ngoài, nhưng nó không thể được truy cập từ bất kỳ lớp ngoài nào.
