1. @Entity

@Entity được sử dụng để chú thích một class là một Entity.

Entity khớp với một bảng lấy theo tên theo thứ tự ưu tiên:

  1. name trong @Table
  2. name trong @Entity
  3. name trong class

Thuộc tính name của @Entity là không bắt buộc. Việc chỉ định rõ name của @Entity cho phép viết ngắn câu HSQL.

Ví dụ trường hợp @Entity chú thích trên class Student không chỉ định rõ phần tử name:

{{ code.code_1 }}

Ví dụ trường hợp @Entity chú thích trên class Student có chỉ định rõ phần name là Student:

{{ code.code_2 }}
