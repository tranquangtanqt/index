Chuyển đổi giữa các kiểu dữ liệu

  1. Trang chủ
  2. Java
  3. Chuyển đổi giữa các kiểu dữ liệu

1. Đổi số thành chuỗi

a. Sử dụng String.valueOf()

String str = String.valueOf(value);

b. Sử dụng toString()

String str = Type.toString(value);

c. Cộng chuỗi rỗng với số

String str = "" + value;

Lưu ý: Khuyên dùng String.valueOf() để tránh lỗi NullPointerException nếu value là null.

2. Đổi chuỗi thành số

a. Sử dụng parseType()

type num = Type.parseType(str);

b. Sử dụng valueOf()

Type num = Type.valueOf(str);

c. Sử dụng decode()

Type num = Type.decode(str);

3. Chuyển đổi từ hệ thập phân sang nhị phân, bát phân, thập lục phân

a. Chuyển đổi từ thập phân sang nhị phân

String str = Integer.toBanaryString(value);

b. Chuyển đổi từ thập phân sang bát phân

String str = Integer.toOctalString(value);

c. Chuyển đổi từ thập phân sang thập lục phân

String str = Integer.toHexString(value);

4. Chuyển đổi từ hệ nhị phân, bát phân, thập lục phân sang hệ thập phân

Type number = Type.parseType(str, 2);
Type number = Type.parseType(str, 8);
Type number = Type.parseType(str, 16);
Type number = Type.valueOf(str, 2);
Type number = Type.valueOf(str, 8);
Type number = Type.valueOf(str, 16);
